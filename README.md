# IT Strategy, Explained Simply

A plain-English study companion for the EMBA "Information Technology Strategy" course
(O'Brien & Marakas, Management Information Systems). All twelve chapters are rewritten
from scratch in simple language, as if you are reading each topic for the first time.

- One hub page (`index.html`) linking to twelve chapter pages.
- Each chapter opens with the whole chapter told as one short story, then explains every
  concept slowly with everyday examples, light diagrams, an "easy to confuse" table, and a
  quick self-check.
- No textbook quotes, no citations, just plain explanation.

Live: https://amsamms.github.io/emba-its-chapters/
